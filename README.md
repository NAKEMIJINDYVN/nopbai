# nopbai

**nopbai** is a simple Python-based application for managing and submitting tasks (assignment management system).

## Features

- Web interface for task submission and tracking  
- Backend logic implemented in Python  
- Lightweight local database (SQLite) for storing tasks and group information  
- Designed for quick deployment and simple usage  

## Technologies

- Python  
- SQLite (local database)  
- Standard Python web framework (Flask / FastAPI)  
- Dependencies managed via `requirements.txt`
